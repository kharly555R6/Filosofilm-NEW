# Filosofilm-NEW
Desarrollo de Filosofilm utilizando React, Nuevos diseños UX/UI e IA integrada

Para iniciar el proyecto los requerimentos son: 
 
 -Tener instalado node.js

Pasos a seguir:
 
 -Abrir nueva terminal (terminal 1 en carpeta frontend)
 -npm install
 -npm start

 -Abrir nueva terminal (terminal 2 en carpeta backend)
 -dotnet build
 -dotnet run

 -El script de la base de datos viene incluido, se debe modificar la cadena de conexión en la 
 linea 13 de backend/Program.cs
